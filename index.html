🧮 DSA — Company-Specific Questions
Amazon: Serialize and Deserialize a Binary Tree
Hint: BFS level-order with null markers. Common in Amazon OA + interviews.
Amazon / Microsoft: LRU Cache — implement using HashMap + DoublyLinkedList
Hint: O(1) get and put. Most interviewers want you to implement from scratch, not use OrderedDict.
Goldman Sachs: Find the Kth Largest Element in a Stream
Hint: Min-heap of size k. Design it as a class with add() method.
PhonePe / Ninjacart: Word Break (DP + Trie approach)
Hint: DP with a dictionary set OR Trie to check valid words. Know both approaches.
Google: Trapping Rain Water
Hint: Two-pointer approach O(n) time O(1) space. Can you explain all three approaches (brute, prefix arrays, two pointer)?
📚 CS Theory — Guaranteed Questions
OS: "What is a deadlock? What are the four necessary conditions?"
Answer: Mutual Exclusion · Hold & Wait · No Preemption · Circular Wait. Know all four with examples.
OOP: "What is the difference between abstract class and interface? When would you use each?"
Answer: Abstract class allows partial implementation + state. Interface is pure contract. Use abstract when sharing code, interface when defining capability.
DBMS: Write SQL to find the second highest salary from an Employee table
Answer: SELECT MAX(salary) FROM employees WHERE salary < (SELECT MAX(salary) FROM employees). Also know the DENSE_RANK() approach.
CN: "What happens when you type google.com in a browser? Walk me through every step."
Answer: DNS resolution → TCP 3-way handshake → HTTP/S request → Server response → Browser rendering. Know each step.
